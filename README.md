## 📖 핵심기능 && 트러블 슈팅

+ **로그인, 회원가입**   
  - JWT를 이용하여 로그인과 회원가입을 구현하였습니다.   
  - 아이디와 닉네임의 중복확인이 가능합니다.      

+ **Todo list, Feedback CRUD**
  - 미들웨어를 이용, 지나간 일자의 체크리스트는 수정이 불가합니다.   
  - 완성한 todo list 개수를 db에 저장, 메인 화면에서 시각효과를 주었습니다.(메인 페이지 색상변화)
